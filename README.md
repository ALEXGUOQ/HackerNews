# HackerNews

A Hacker News reader iOS app written in Swift.

![Main View](http://idzr.org/xd1k)

## Getting Started

1. `git clone https://github.com/amitburst/HackerNews.git`
2. `cd HackerNews`
3. `pod install`
4. Open `HackerNews.xcworkspace` in Xcode 6.1 or higher
